# Data Quality & Validation

This folder contains SQL-based validation checks that ensure
the reliability and accuracy of curated datasets.

## Quality Checks Implemented
- Duplicate key detection
- Null or invalid foreign keys
- Negative or inconsistent metric values
- Sanity checks on pricing and revenue fields

## Scope
Data quality checks focus primarily on the curated (Gold) layer,
where trusted analytics datasets are exposed to downstream users.
